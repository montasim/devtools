Component Architecture

How the pieces fit together

Component Hierarchy

EditorPane (Orchestrator)

• Manages state for both editors
• Handles file uploads
• Coordinates diff computation
• Passes props to children

JsonEditor (x2)

• CodeMirror instance
• Shiki syntax highlighting
• Real-time JSON validation
• Error display with line/column

DiffPanel

• Unified diff view (+/-)
• Line numbers
• Color-coded changes
• Toggle-based filtering

useJsonDiff (Hook)

• Diff computation logic
• Applies toggle settings
• Handles formatting
• Returns diff result

Data Flow

User Input → JsonEditor → EditorPane State → Compare Click → useJsonDiff → DiffPanel

File Structure
