Validation States

Empty

No content → Compare disabled, show "Enter JSON to compare"

Invalid

Parse error → Compare disabled, show inline error with line/column

Valid

Parse succeeds → Compare enabled, green check indicator

Error Scenarios

Scenario Behavior
Invalid JSON paste Show error, highlight error line, disable Compare
Invalid JSON file upload Toast error, don't populate editor
Huge file (>10MB) Warning toast, still load but may be slow
Diff fails unexpectedly Error message in DiffPanel, retry button

File Upload Handling

  • Accept: .json files only (initially)
  • Size limit: 10MB warning, 50MB hard limit
  • Encoding: UTF-8 only, show error if invalid
  • Validation: Parse on upload, reject if invalid with clear error
  • UX: Drag-and-drop zone + file input button
